# Git Secrets Guard (Gitleaks Pre-commit Hook) 🛡️

Цей репозиторій автоматизує перевірку коду на наявність секретів (API-ключів, паролів, токенів) перед кожним комітом. 

Скрипт самостійно визначає вашу ОС (Windows, macOS, Linux), завантажує відповідну версію **Gitleaks** і встановлює її локально в репозиторій.

---

## 🚀 Встановлення в один крок

Виконайте команду у кореневій папці вашого проєкту:

```bash
curl -sSL https://raw.githubusercontent.com/maxbet111/pre-commit-hook/main/install.sh | bash

# Активація та керування
Після встановлення хук потрібно активувати для конкретного репозиторію. Це дозволяє уникнути небажаних перевірок там, де вони не потрібні
Увімкнути захист:
git config hooks.gitleaks.enabled true

Тимчасово вимкнути захист:
git config hooks.gitleaks.enabled false
