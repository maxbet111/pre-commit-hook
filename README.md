# Git Secrets Guard (Gitleaks Pre-commit Hook) 🛡️

Цей репозиторій містить Python-скрипт для автоматичної перевірки вашого коду на наявність секретів (API-ключів, паролів, токенів) перед кожним комітом. 

Скрипт використовує **Gitleaks**, автоматично встановлюючи його під вашу операційну систему (Linux, macOS, Windows).

---

## 🚀 Швидке встановлення

Щоб встановити хук у будь-який локальний Git-репозиторій, виконайте команду у кореневій папці вашого проєкту:

```bash
curl -sSL [https://raw.githubusercontent.com/ВАШ_ЛОГІН/pre-commit-hook/main/install.sh (https://raw.githubusercontent.com/ВАШ_ЛОГІН/pre-commit-hook/main/install.sh) | bash
